# TransformerAQPredict

A Transformer-based regression model to predict air quality using structured tabular data from OpenAQ.

## Description
This project implements a deep learning model based on Multi-Head Attention to forecast air pollution levels using numerical and categorical features from the OpenAQ dataset.

## Dataset
Dataset: [OpenAQ](https://public.opendatasoft.com/explore/dataset/openaq)

To use:
1. Download the dataset from OpenAQ in `.csv` format.
2. Place it in the project folder as `openaq.csv`.
